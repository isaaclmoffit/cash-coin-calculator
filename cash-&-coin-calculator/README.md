# Cash & Coin Calculator

This is a simple Lua program I wrote that runs via the terminal. It prompts the user for quantities of bills and coins, then outputs a formatted text file with calculated totals.

I made it because I count my physical money, and I wanted to automate the documenting/calculating step as a fun coding project.

## Currently, it features:
    - input validation
    - file generation
    - automatic calculations
    - automatic formatting

## This is the first iteration. I intend to add:
    - the ability to run the program from anywhere on the computer
    - the ability to select a location to create the text file
    - anything else I decide to improve on

Requires Lua to be installed on the system.

Run from the directory that contains the program:

```bash
lua cash-coin-calculator.lua
```
